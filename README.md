# Blog Spa - Vanilla JavaScript

### Getting started

1. Clone this repo 

```shell
git clone https://github.com/Front-End-Focused/blog-spa-js.git
# or
git@github.com:Front-End-Focused/blog-spa-js.git
```
2. `cd blog-spa-js`
3. `npm install`
4. `npm run dev`
